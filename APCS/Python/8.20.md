初階版的泡沫排序法
時間複雜度: n\**2

``` python
# 0  1  2   3  4
a = [5, 4, 3 , 2, 1]    #len(a)
for i in range(len(a)-1):   #掃n-1遍
    for j in range(len(a)-1):   #n-1 => n**2 
        if a[j] > a[j+1]:
            tmp = a[j]
            a[j] = a[j+1]
            a[j+1] = tmp
```

優化

時間複雜度: n**2

``` python
a = [5, 4, 3 , 2, 1]    #len(a)
for i in range(len(a)-1):   #掃n-1遍  0 1 2 3 
    for j in range(len(a)-1-i):   #n-1 => n**2 
        if a[j] > a[j+1]:           # 0 3  - 0
            tmp = a[j]              # 0 2  - 1
            a[j] = a[j+1]           # 0 1   - 2
            a[j+1] = tmp
```

## 選擇排序法

``` python
# x宏
ans=[]
a=list(map(int,input().split()))
ans.append(a[0])
del a[0]
while True:
    if len(a)==0:
        break
    for j in range(len(ans)):
        if ans[j]>a[0]:
            ans.insert(j,a[0])
    del a[0]
print(ans)
```

``` python
def SelectionSort(arr):
    for i in range(len(arr)):
        minv = 10**9
        index = -1
        for j in range(i, len(arr)):
            if minv > arr[j]:
                minv = arr[j]
                index = j
        tmp = arr[i]
        arr[i] = minv
        arr[index] = tmp
    return arr
```
