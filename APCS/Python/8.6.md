## 遞迴速寫

如果某一個問題，每一個階段的問題解法規則都是一樣的

ex.

1 + 2 + 3 ..  + 100



f(n): n 階段的結果

f(100): 5050

1 + 2 + 3 +...+ 50

f(50) = 1 + 2 + 3 +...+ 50



f(n) = f(n-1) + n

邊界 1



遞迴關係式

f(n) = f(n-1) + n



邊界

f(1) = 1

``` python
def f(n):
    if n == 1:
        return 1
    return f(n-1) + n
print(f(100))
```

---

階層

1*2*3*..*n

f(n) = f(n-1)*n

f(100) = 1*2*3*..*100 

f(99) = 1*2*3*..*99



遞迴關係式

f(n) = f(n-1) * n



邊界

f(1) = 1

``` python
def f(n):
    if n == 1:
        return 1
    return f(n-1) * n
print(f(100))
```
